# PetMatch — 제품 요구사항 문서 (PRD)

> 버전: 1.0 | 기준 문서: PetMatch_개발명세서_v2.pdf | 작성일: 2026-04-27

---

## 1. 제품 개요

PetMatch는 반려동물 보호자가 산책 시간대·동선·카페 선호 같은 **라이프스타일 필터**와 **규칙 기반 호환성 점수**를 통해 진짜 잘 맞는 이웃 보호자를 찾는 **반려동물 중심 소셜 웹 앱**이다. Tinder와 같은 스와이프 UX를 유지하되, 단순 외모 매칭이 아니라 "같은 공원, 같은 시간, 같은 루틴"을 가진 사람을 연결하는 데 초점을 맞춘다. Next.js로 개발하여 Vercel에 배포하고, Supabase로 인증·데이터·실시간 채팅·이미지 저장을 처리한다.

## 2. 타겟 사용자 & 핵심 사용 시나리오

### 타겟 사용자
- 반려동물을 키우는 20~40대 보호자
- 정기적인 산책 파트너·플레이데이트 상대를 찾는 보호자
- 같은 시간대, 같은 동네 공원에서 반복적으로 만날 이웃을 원하는 보호자
- 반려동물 동반 카페·펫 친화 장소 정보를 교류하고 싶은 보호자

### 핵심 사용 시나리오
1. **신규 가입** → 보호자·반려동물 프로필 생성 → 목적 태그 선택(산책친구·플레이데이트 등) → 라이프스타일 설정(산책 시간대·자주 가는 공원·카페 선호)
2. **Discovery 탐색** → 호환성 점수와 라이프스타일 매칭 표시가 된 카드를 스와이프 → 마음에 들면 Like
3. **매칭 성사** → 양쪽 Like → 매칭 팝업 → 첫 만남 안전 체크리스트 확인 → 실시간 채팅
4. **약속 잡기** → 채팅 중 산책 약속 카드 전송 → 상대방 수락 → 만남
5. **후기 남기기** → 만남 후 펫티켓 체크리스트 후기 작성

| 13 | **산책 약속 카드** | 채팅 중 날짜·시간·장소 템플릿 전송·수락 |

## 4. 각 기능의 수용 기준 (Acceptance Criteria)

### 기능 1 — 인증
- 이메일·비밀번호로 회원가입·로그인이 된다
- Google 소셜 로그인이 된다
- 로그인 상태가 새로고침 후에도 유지된다
- 로그아웃하면 보호된 페이지에 접근할 수 없다

### 기능 2·3 — 프로필
- 보호자와 반려동물 정보를 모두 입력해야 Discovery에 진입할 수 있다
- 사진은 최대 5장까지 업로드되고, Supabase Storage에 저장된다
- 프로필 수정이 가능하며 수정 후 카드에 반영된다

### 기능 4 — 목적 태그 온보딩
- 첫 가입 시 목적 태그 선택 화면이 자동으로 뜬다
- 복수 선택이 가능하고, 적어도 1개는 선택해야 다음으로 넘어간다
- 선택한 태그가 내 카드에 표시된다
- 나중에 프로필 설정에서 변경할 수 있다

### 기능 5 — 신뢰 뱃지
- 예방접종·중성화·성향·입마개 필요 여부·개 친화도·사람 친화도 6가지를 입력할 수 있다
- 입력한 뱃지가 상대방 카드 하단에 아이콘으로 표시된다

### 기능 6 — 라이프스타일 설정
- 산책 시간대를 7개 슬롯(새벽/아침/오전/점심/오후/저녁/밤) 중 복수 선택할 수 있다
- 자주 가는 공원·장소를 최대 3곳 텍스트로 등록할 수 있다 (지도 핀은 Post-MVP)
- 카페 선호를 "좋아함·괜찮음·선호 안 함" 중 선택할 수 있다

### 기능 7 — 호환성 점수
- Discovery 카드 상단에 "궁합 OO%" 뱃지가 표시된다
- 80% 이상 초록, 60~79% 주황, 60% 미만 회색으로 색상이 다르다
- 카드를 탭하면 항목별 점수 상세(에너지·성격·시간대·체급·카페)를 볼 수 있다
- 점수 계산에 필요한 정보가 없으면 "정보 부족" 표시 후 입력을 유도한다
- 점수 계산은 서버 없이 브라우저에서 실행된다

### 기능 8 — Discovery (스와이프)
- 카드를 오른쪽으로 드래그하면 Like, 왼쪽으로 드래그하면 Pass 처리된다
- 드래그 방향에 따라 초록 하트·빨간 X 오버레이가 나타난다
- 하단 버튼으로도 동일한 동작이 가능하다
- 내가 이미 Like/Pass한 카드는 다시 나타나지 않는다
- 볼 카드가 없으면 "주변에 더 이상 카드가 없어요" 메시지가 표시된다

### 기능 9 — 필터
- Discovery 화면에서 필터를 열고 조건을 설정하면 카드 목록이 즉시 바뀐다
- 필터 설정은 새로고침 후에도 유지된다

### 기능 10 — 상호 매칭
- 두 사람이 모두 Like를 누르면 매칭 팝업이 표시된다
- 매칭 목록 탭에서 내 모든 매칭을 볼 수 있다

### 기능 11 — 첫 만남 안전 체크리스트
- 매칭 성사 후 채팅방에 안전 체크리스트 카드가 자동으로 삽입된다
- 항목(예방접종 확인·장소 선택·친화도 확인·중성화 여부)을 모두 체크해야 약속 카드를 보낼 수 있다

### 기능 12 — 1:1 실시간 채팅
- 매칭된 상대와 텍스트 메시지를 실시간으로 주고받을 수 있다
- 사진을 전송할 수 있다
- 상대방이 메시지를 읽으면 읽음 표시가 된다
- 새 메시지가 오면 브라우저 탭 타이틀에 알림이 표시된다

### 기능 13 — 산책 약속 카드
- 채팅 중 약속 카드 버튼을 눌러 날짜·시간·장소를 입력하고 전송할 수 있다
- 상대방이 수락하면 카드 상태가 "확정됨"으로 바뀐다

### 기능 14 — 신고·차단
- 채팅 또는 카드 화면에서 신고·차단을 할 수 있다
- 차단한 사용자의 카드는 Discovery에 나타나지 않는다

## 5. 범위 밖 (Out of Scope — MVP 기준)

- 모바일 앱 (iOS/Android) — 웹 앱만 개발 (모바일 브라우저에서 반응형으로 사용 가능)
- Push 알림 (FCM) — 브라우저 탭 타이틀 알림으로 대체
- 실제 지도·GPS 핀 등록 — 장소를 텍스트로 입력하는 방식으로 대체
- 결제·프리미엄 구독
- AI/LLM API 호출 (Claude API) — 규칙 기반 점수로 대체
- 관리자 대시보드
- 이메일 마케팅·푸시 캠페인
- 카카오 로그인 (Supabase 기본 제공 범위 밖)
- Apple 로그인 (Post-MVP)
