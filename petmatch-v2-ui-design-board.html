Design System

앱 전체는 병원이나 일반 데이팅 앱이 아니라, 편안하고 세련된 한국식 펫 카페 경험처럼 느껴져야 합니다.

UI Mood

둥근 카드, 부드러운 그림자, 영수증형 약속 카드, 카페 메뉴판 같은 칩 선택 UI를 사용합니다.

Core Components

Pet Card, Lifestyle Chip, Trust Badge, Filter Bottom Sheet, Swipe Action Row, Safety Checklist, Chat Receipt Card를 우선 컴포넌트화합니다.

Motion Rules

스와이프는 각도에 따라 회전하고, LIKE는 Matcha Heart, PASS는 Coral X, Match는 포토부스 팝업으로 표현합니다.

Premium Tone

프리미엄은 강한 결제 압박보다 “더 좋은 산책 친구와 카페 메이트를 찾는 멤버십”처럼 표현합니다.

Key Screens

와이어프레임의 MVP 우선순위에 맞춰 실제 개발자가 React Native 화면으로 옮기기 쉬운 구조로 잡았습니다.

Implementation Handoff

다음 개발 단계에서는 이 보드의 색상 토큰, 카드 레이아웃, 칩/뱃지 스타일을 `apps/mobile`의 공통 theme 파일과 재사용 컴포넌트로 옮기면 됩니다. 우선 구현 순서는 Onboarding, Pet Profile, Discovery, Discovery Filter, Safety Checklist, Match Popup, Chat입니다. 이후 Places Map, Group Walks, Profile Trust, Premium을 확장하면 됩니다.
